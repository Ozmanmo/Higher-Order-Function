map : This function takes an array and a function as arguments, applies the function to each element of the array and returns a new array with the modified elements.

filter : This function takes an array and a function as arguments, applies the function to each element of the array, and returns a new array with only the elements that pass the test.

reduce: This function takes an array, a function and an initial value as arguments, applies the function to each element of the array and reduces the array to a single value.

forEach : This function takes an array and a function as arguments applies the function to each element of the array and returns nothing.
